# Usage

Clone the boilerplate

```bash
$ git clone https://github.com/vuepress-boilerplate
```

Now, you may tweak it how you want it!

To remove the reference docs and the `.git` folder, run

```bash
$ npm run docs:clean
# or
$ yarn docs:clean
```
